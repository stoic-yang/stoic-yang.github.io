### **Structs**

Define a struct:

```cpp
StanfordID issueNewID() {
    StanfordID id;
    id.name = "Jacob Roverts-Baca";
    id.sunet = "jtrb";
    id.isNumber = "6504417";
}
```

与下面定义是等价的，可以直接用逗号分隔开

```cpp
StanfordID id { "Jacob Roverts-Baca", "jtrb", "6504417"};
```
