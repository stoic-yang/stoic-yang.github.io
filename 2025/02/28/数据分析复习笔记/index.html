# 支持向量机

[!PDF|] 数据科学技术与应用 (第 2 版) ocr 带书签，p.109

支持向量机 (Support Vector Machine, SVM) 是基千数学优化方法的分类学习算法。它的基本思想是将数据看作多维空间中的点，求解出一个最优的超平面，从而将两种不同类别的点分割开来。

# 聚类
