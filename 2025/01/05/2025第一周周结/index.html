# 碎碎念（不重要，可以跳过）

第一周就过得挺烂的，哈哈，不过已经调整过来了，元旦放假，打算休息一天，结果熬夜了，接着 2 天状态也是炸裂，搞的是我之前刚写完一篇关于如何避免娱乐性摆烂的文章，然后我还是没有抵抗力，心理暗示有很大关系。摆的那天在推人狼村之谜，这种轮回的剧本还是很戳我的，只是四周目感觉有点烂尾，总的来说也是非常优秀的剧本，亮点主要在男主上，双商极高，太会说话了，动摇了我以前固执的一些观念，还让我想谈恋爱了，会说话的人真的有魅力呀，房实阳明（男主）跟村里人的对话都可以作为锻炼情商的教材了，有时间得好好练练口才了。太牛了，诶多 man

# 信息蒸馏

其实我之前就接触过这个方法（或者说思想），只是当时没有意思到处理每天信息的重要性。我是一直被个性化推荐迫害的，经过几次摆烂后，我的 b 站首页已经是不堪入目了，知乎推荐也是，这样下去，我以后就很难接触到高质量的信息，每天被低质量的信息毁灭了。

一个好的办法，之前偶然刷到的，我突然想起来了 ———— 关注高质量的博主，然后顺着他们的关注和点赞去关注其他博主。

这一点在 QQ 群群聊消息上也是适用的，寻找群友推荐的优秀文章，然后关注作者，再关注作者关注的人，这可以快速扩大自己的高质量信息来源，而以后就再也不把自己暴露在个性化推荐上，只看自己关注的高质量博主的文章以及收藏夹，这样就是摆烂，看到的东西都是很有价值的，不会陷入到奶头乐的囚笼。

我今天就是开始实践这个方法，刷屏带来的收益显著的提高了，看到这样高质量的信息还不会像低质量的信息给我带来一堆负面影响 ———— 注意力下降，想玩游戏，累

# 娱乐方式的一个可能性平替方案

另一个值得一提的是我似乎找到了娱乐化休息的平替 ————b 站的公开课，我指的不是计算机公开课，而是哲学、经济学（比如博弈论）这类公开课，看这类公开课也可以达到休息的目的，而且可以学到很多有意思的东西，扩展我的知识面。

# 几个有意思的点

星爷的知识库看到几个很有意思的想法

  • 时间表，是为了，让你有一个你想要的一天
  • 如果需要和厉害的人争论一些东西，你需要去谨慎的组织你们的观点，盲目的争论很可能会被看起来很蠢
  • 如果我全身心投入到我想要的事情上，我的生活会如何？
  • 你的心脏每秒钟都在跳动，那是你与死亡的搏斗

# Review

JavaWeb 接口开发
蒸馏信息流的一个方法
上手 cline 和 cursor
了解 deepseek 接口的使用 —— 给了沉浸式翻译

# Reflection

这周烂的时间很长，摆烂时间频繁而且长，没有学到很多东西，下周就会好很多

虽然我通过翘课给自己留下巨量的自由时间，但我每天浪费的时间非常多，多到让我无法统计，我浪费太多时间了，JavaWeb 两周都没学完，太丢人了，接下来我会好好珍惜时间的😭

虽然过去一个月每天都在 all in 计算机学习，但是实践看来，每天有效的学习时间也就 4 小时罢了，不如分时间弄学业，试试插班生什么的，争取保研吧，211 的 title 确实是不够用的（对我来说）

# Planing

  1. cline 的出现改变了我原有的一些计划，让我意识到搓出项目的壳子也许并不是一件很难很费时的事情，也许我一天就能办到，现在我的目标是在下一周就把 JavaWeb 结课 + 搓出这个壳子
  2. 抽出一个流的时间处理学业问题
  3. 空余时间研究 anki 的用法，为以后背单词做准备，不背单词不能自定义处理单词本这一点就让我彻底放弃它了
