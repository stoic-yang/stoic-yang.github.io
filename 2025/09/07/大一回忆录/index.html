大一下回忆录

郁孤台下清江水，中间多少行人泪。

非常迟到的大一下总结复盘，我这里的大一下复盘涵盖的范围与一般的人所定义的有所不同，主要是源自于我校奇怪的学期制，其次是我希望这样定义，不然我的大一下很难看了，所以大概是 3-9 月的回顾与记录。

大一下简直是地狱开局，冬季学期的考试给了我当头一棒，大物 1 的爆炸直接让我的绩点下降了 0.5，让我本来就不太好看的绩点直接变得没法看，此事在我的知乎亦有记载，总之是破了大防，现在想来还是当时没有积累好期末冲刺的经验，自己把自己的绩点玩炸了。

虽然学校给英语考试分级了，但为什么 C 级的英语这么难？上海本地的考生基本都是 A 级，考试的成绩也基本是 A。而英语 C 级的人英语绩点还没有见到过大于 3.0，而且几乎全部英语 C 级的人都是压线过的。你外国语学院出的什么卷子？你分的这个等级有什么意义？大物大题一点步骤分都不给的吗？为什么打分这么低？

不过我也没有放弃自救，即使在破防的状态下也在一天内调整了过来，继续做为下一周做计划，记录自己每一周的成功。在春季的前 3 周，我把课余的时间拿去写苍穹外卖了，最后 curd 写的太无聊了，前面有跟着写，后面就没有跟着写，只是把课过了一遍。

在第三周写完苍穹外卖后，我又陷入了一种迷茫的状态，当时没有想过其实自己可以坚持学下去，把剩下的后端东西学了，把 leetcode 写了，准备暑假实习，后面我发现这条路是可行的，但是当时确实是没有信心去这么做。我毕竟还需要维持绩点去分流。

那么写完苍穹外卖以后学什么呢？我当时甚至想花两个月去考插班生，印象中是我是接着学了 61 c 的一点点内容，并且在 leetcode 上刷了几天题目，学了一点滑动窗口。春季学期的主基调仍旧是卷绩点，61 c 又太硬，所以学了一点点，连 proj 1 都没写就断了。那几周我的生活状态大概是，什么都想做，但什么都做不起来，课内的东西又总是把我的心力搞枯竭，晚上很多时间在打游戏，什么 2077，甚至一度成为素学家，一周一大半的时间在摆烂，剩下的时间在想怎么解决自己摆烂的问题，这段时间里我有写和看很多知乎里面有关自制力的思考，并且重新去读了福格行为模型。

春季 4-7 周我过得晕头转向的，一会插班生、一会 61 c、一会 leetcode、一会蓝桥杯、一会熟悉 macos，一会忙课内卷绩点，一会应付课内作业压力（尤其是工图）。没有一个明确的目标和主线，所以一整个月都是忙忙碌碌有碌碌无为。当时其实就是没有想清楚，自己想要什么，并且对自己失去太多信心了。不相信自己还能做学业以外的事情，担心自己学了这个那个，结果绩点爆炸，但实际上也没有怎么卷绩点，所以在担忧受怕中度过了一个月。

7-11 周，考试周，我甚至周结都找不到了，不记得当时是太忙没写还是没有状态没写了，但最后春季的考试成绩在我的预期线上，所以这个月虽然没有学到什么，但是还是可以接受的。经过两个学期的失败，终于是摸到了一点怎么卷绩点的秘诀。

12 周，考试结束后的一周，我玩了几天 neovim，学了两周的 CS 106 L，做到了 assignment 4，中途参加大创，花了两天时间弄了一个非常简陋的小程序前端 UI，写的真的是非常丑，ai 调教的水平有待提升，当时时间紧张，也确实只有上 ai 才有机会在短时间内实现出来了。但是好像是觉得我一个 java 选手学 C++没啥太大意义，所以又跑去 java 复健了，可以说当时好几个月没有写过 java 了。

这个时候已经是夏季学期的第二周了，一个炸弹来了。

由于我是通过国家专项计划上的学，而分流规则中高考成绩占的比例非常大，直接导致我的排名爆降 40%，什么概念呢，大概就是从专业随便选到计算机学院可能都去不了，一想到我 tmd 学了一年的计算机，最后没有能够分流去计院，我就感觉很幽默，心直接死了（当时我是这么认为的），一度考虑过大二转专业，或者辅修计算机，卷保研跨保这些东西。

不过还是很幸运啊，或许是计算机这两年热度下降了，我还是得以去到一个计算机类专业，还是分流到了计算机学院。随后我还惊奇地发现自己在专业内的排名还不错，我当时？，嘛玩意，分流后排名上升了 70%，我了个豆哇，我怎么还突然有希望保研了，一下子信心倍增，原本打算彻底放弃绩点的，夏季学期前半段时间疯狂摸鱼摆烂，后半段疯狂卷大作业，虽然后面看来卷这个这么水的大作业很小丑是了，但是在小组中认识了一个非常厉害的队友，应该是得到了大佬的认可，最后大佬拉我一起组寝室了。

后面很顺利地组好了寝室，很顺利地加入到了集训队，很顺利地度过了短暂 8 周的集训，得到一次去今年区域赛的资格。而现在正是集训的结尾，集训最后一周的周四。

五月份的周结结尾，我这样写道

更糟糕的是冬季绩点爆炸，自信心进一步降低，在前四周做完苍穹外卖后就基本没有写过代码了，茫然、后悔，插班生 g 了，蓝桥杯 g 了，绩点 g 了，acm 也 g 了，所有的路好像都对我关闭了，茫然、后悔，接着便是摆烂到现在。

从后往前看，过去以为的失败似乎并不尽然。

那是 6 月 15 号的下午，我和 zlh 刚结束一场能够与院长直接面对面沟通的交流会，在回去的路上，我跟 zlh 说，高考失意的时候，我感觉自己完蛋了，彻底失败了，但是大一刚开学的就在学 cs 50 的我感觉未来可期，人可能就是这样，在一次失败后，感觉自己无药可救了，但是在如果以过一段时间的眼光再来看现在的自己，一定会觉得自己现在仍旧是充满希望的。

现在的我，往前看那段我大一最担忧受怕、焦虑痛苦的时光，又看到现在自己，觉得当时的担忧焦虑实在是被自己过分强化了，自己远比自己想象中做的好，自己远比自己想象中有希望。

当时觉得所有的路都被封死了，然而并不尽然，只要现在开始想，好好地想自己想做什么，自己要做什么，然后马上行动起来，锁住门的链条或许就突然消失不见了，这时候我才发现，把门锁上的是其实是我自己。

随着我不断地学习和思考，我的观念也发生了很多变化，其一是不拧巴了，我不再为自己没法按照预期学习和摆烂感到极度内疚了，因为我知道这是不合理的生活模式设计照成的，我不再思考为什么自己做不到别人那么自律，而花时间去思考怎么优化自己的设计的生活模式。另外是，我的目标从追求财富自由（也许？），变为了最大化地体验人生，比如说 acm 训练很累，但这就是 acm 的味道，just for fun？所以我即使很坐牢，但还是享受集训的过程，但我的队友没法做到这样，集训大部分时间都不在状态，所以集训期间我的训练量几乎是他的两倍。虽然我感觉自己集训也没有做到该有的认真（

所以最后结束来，发现自己大一做的还不错，绩点能看，课外的东西学了也不少，有了一套比较成熟的工作学习流，最后也交到了不少的朋友，活着挺滋润的。

这段时间有一直刷到很多大一、大二失败经验总结，每次看完之后，我也跟着情绪变得低落了，竟然也有想着把自己从低谷中走出来的经历分享出来，鼓励那些像过去的我，看到这里的朋友，也可以说是说给未来的自己，不要把当下的状况想的那么糟糕，不要自暴自弃，人的精神面貌是可以在 3 个月的时间里发生巨大改变的， 3 个月也可以让一个计算机学生的知识量翻一倍，通过说一堆别人听不懂的专业名词吹逼来找回自信（bushi）
